# Deploy

> **Operator role required.** All tasks in this section require GitHub organisation owner access and the ability to run containers on a server.

| Guide | What it covers |
|---|---|
| [Create the metadata repository](create-metadata-repository.md) | Set up the `.reporoller` repository that holds all configuration |
| [Register the GitHub App](register-github-app.md) | Create the service-identity App used to create repositories |
| [Register the GitHub OAuth App](register-oauth-app.md) | Create the App used for user authentication in the web UI |
| [Deploy with Docker Compose](deploy-with-docker.md) | Run both containers with Docker Compose |
| [Configure branding](configure-branding.md) | Customise logo, name, and colours |

For a guided first-time deployment, follow the [Set up RepoRoller for your organization](../../tutorials/set-up-reporoller.md) tutorial.
